# Job Application Tracker

Track job applications, interview progress, and follow-ups in one place.

## What this project is for

This repository is for organizing a job search workflow. It can be used to keep a record of:

- companies and roles
- application dates and links
- status updates
- interview notes
- follow-up reminders

## Getting started

Clone the repository and open it in your preferred editor:

```bash
git clone <repository-url>
cd job-application-tracker
```

## Usage

Create an entry for each application and update it as the process changes.

Recommended fields:

- Company
- Role
- Location
- Date applied
- Status
- Contact person
- Notes

## Suggested workflow

1. Add a new application after applying.
2. Update the status when you hear back.
3. Record interview details and feedback.
4. Add reminders for follow-ups.

## Example statuses

- Applied
- Screening
- Interviewing
- Offer
- Rejected
- Withdrawn
